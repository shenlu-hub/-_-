# 实验目的
1. 掌握Java中抽象类和抽象方法的定义； 
2. 掌握Java中接口的定义，熟练掌握接口的定义形式以及接口的实现方法

1. 在博士研究生类中实现各个接口定义的抽象方法;

5. 根据输入情况，要在程序中做异常处理。
